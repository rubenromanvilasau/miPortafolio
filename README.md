 ## Variables de entorno

  DB_URI= link mongodb
  EMAIL= smtp email
  EMAIL_PASSWORD= email password
  
 ## Correr en local
  Para correr en local primero configurar variables de entorno, luego correr el comando npm install, y posteriormente ingresar a localhost:3000
